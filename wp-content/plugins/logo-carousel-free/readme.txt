== Description ==
Logo Carousel Free is a fully responsive slider plugin to display logos of your clients, partners, sponsors, affiliates etc in a wonderful slide effect or carousel. It allows you to add unlimited logo images in a slider with logo carousel shortcode. You can add this shortcode `[logo_carousel_free id="01"]` multiple times in anywhere to your WordPress website.

== Installation ==

Installing this plugin as regular WordPress plugin.

After install, you are ready to use logo carousel in your post, page, widget or anywhere you like. How to create logo carousel? lets see.


1. Upload the folder “logo-carousel-free” to "/wp-content/plugins/" '
2. Activate the plugin through the "Plugins" menu in WordPress .
4. Add logo going to wp-admin -> Logo Carousel Free -> Add New Logo.
5. Call shortcode to your WordPress editor directly, by using 
`
[logo_carousel_free id="01"]
`
Call shortcode to your php template file using 
`
<?php  echo do_shortcode('[logo_carousel_free id="01"]’); ?>
`
